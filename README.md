PrepWise AI is a full-stack AI-powered mock interview platform designed to help students and job seekers practice technical interviews in a realistic environment.

The platform generates AI-based interview questions, evaluates candidate answers, provides detailed feedback, tracks interview performance, and simulates real interview pressure using a countdown timer.

Features
AI-generated interview questions
AI-based answer evaluation
Question-wise feedback & analysis
Interview timer simulation
Performance analytics dashboard
Authentication & protected routes
User-specific interview history
Modern responsive UI
